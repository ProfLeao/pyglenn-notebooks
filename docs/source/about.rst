=====
About
=====

**pyglenn** is a lightweight, dependency-free thermochemical properties calculator
developed by Dr. Reginaldo G. Leão Jr. at GESESC / IFMG.

Source code: https://github.com/ProfLeao/pyglenn

License
=======

Refer to the repository for licensing information.

Reproducing
===========

The notebooks are committed with their outputs already executed. To re-run them,
install the dependencies and execute each notebook in order.

Requirements
============

.. code-block:: bash

   pip install pyglenn
   pip install numpy pandas matplotlib scipy
