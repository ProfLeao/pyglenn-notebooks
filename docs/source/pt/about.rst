=====
Sobre
=====

**pyglenn** é um calculador de propriedades termoquímicas leve e sem dependências
externas, desenvolvido pelo Dr. Reginaldo G. Leão Jr. no GESESC / IFMG.

Código-fonte: https://github.com/ProfLeao/pyglenn

Licença
=======

Consulte o repositório para informações sobre licenciamento.

Reproduzindo
============

Os notebooks são commitados com suas saídas já executadas. Para reexecutá-los,
instale as dependências e execute cada notebook em ordem.

Requisitos
==========

.. code-block:: bash

   pip install pyglenn
   pip install numpy pandas matplotlib scipy
