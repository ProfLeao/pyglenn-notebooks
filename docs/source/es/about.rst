==========
Acerca de
==========

**pyglenn** es un calculador de propiedades termoquímicas ligero y sin dependencias
externas, desarrollado por el Dr. Reginaldo G. Leão Jr. en GESESC / IFMG.

Código fuente: https://github.com/ProfLeao/pyglenn

Licencia
========

Consulte el repositorio para información sobre la licencia.

Reproduciendo
=============

Los cuadernos se suben con sus salidas ya ejecutadas. Para reejecutarlos,
instale las dependencias y ejecute cada cuaderno en orden.

Requisitos
==========

.. code-block:: bash

   pip install pyglenn
   pip install numpy pandas matplotlib scipy
